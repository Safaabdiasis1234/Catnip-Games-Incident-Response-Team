# TheHive
- Case management and task follow-up platform, helpful for easy live collaboration in a SOC. 
- Import events from SIEM tools 
- Dashboard for aggregating cases and metrics
- Creating Cases
  - Severity (low, medium, high, critical)
  - TLP (Traffic Light Protocol) for disclosure level
  - PAP (Permissible Actions Protocol) for sensitivity of response actions to avoid alerting attackers
  - Tags
  - Description
  - Tasks
  - Observables
  - TTP (Tactics, Techniques, and Procedures)

# MISP (Malware Information Sharing Platform)
- Threat Intelligence
- Use in reverse engineering, security investivations, intelligence analysis, risk analysis
- IOC database, data can be shared across instances
- Dashboard
  - Live Feed 
  - Event Actions
    - Add Event
      - Distribution
    - Attributes

# Cortex
- Automated Response & Detection
- Dashboard with custom widgets for easy monitoring
